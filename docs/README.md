# Documentación

La documentación del Agentic Repo Framework se organiza en tres capas
independientes y autosuficientes, una por audiencia. Cada capa deriva del
conocimiento verificado del repositorio, no de las demás.

## Capas

### Para agentes de coding

Compacta y modular, cargable bajo demanda. Suficiente para localizar y modificar
el código con seguridad sin depender de las otras capas.

- [docs/agent/](agent/README.md) — índice de la capa para agentes.

### Para desarrolladores y mantenedores

Contexto técnico completo para comprender, mantener y ampliar el proyecto.

- [docs/development/](development/README.md) — índice de la capa de desarrollo.

### Para usuarios

Guía de uso: instalar, sincronizar y aprovechar las skills del framework.

- [docs/user/](user/README.md) — índice de la capa de usuario.

## Metodología

Los principios de mantenimiento documental viven en
[documentation-methodology.md](documentation-methodology.md). Es un archivo
**gestionado** por `agentic-sync` (declarado en `.agentic-framework.json`) y
forma parte del contrato común del framework.
