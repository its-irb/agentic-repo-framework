# Documentación para desarrollo

Capa técnica completa para comprender, mantener y amplir Agentic Repo Framework.
Es autosuficiente: no requiere consultar `docs/agent/` para trabajar.

## Documentos

- [architecture.md](architecture.md) — intención, arquitectura, estructura,
  flujos y decisiones de diseño.
- [skills-system.md](skills-system.md) — sistema de skills en profundidad:
  implementación común, wrappers, añadir skills, cuándo modificar cada fichero.
- [sync-model.md](sync-model.md) — modelo de sincronización completo: tipos de
  skills, reglas, CLI de `agentic-sync.py`, estados, manifest y lockfile dual.
- [operations.md](operations.md) — operación y mantenimiento: añadir
  skills/wrappers, sincronizar, actualizar documentación, preparar el entorno.

## Metodología

Los principios de mantenimiento documental viven en
`docs/documentation-methodology.md` (archivo gestionado por el framework).
