# Documentación de uso

Guía para utilizar Agentic Repo Framework: instalarlo en un repositorio,
sincronizarlo y aprovechar sus skills. Es autosuficiente: no requiere consultar
documentación técnica.

## Documentos

- [getting-started.md](getting-started.md) — qué es el framework, para quién es,
  requisitos e instalación paso a paso.
- [sync-guide.md](sync-guide.md) — cómo sincronizar el framework en un
  repositorio consumidor: `--plan`, `--apply`, `--force`, conflictos y lockfile.
- [skills.md](skills.md) — skills core disponibles y cómo añadir skills propias
  (Repo o Personales).
